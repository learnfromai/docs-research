- **Transition**: Moved from support to development to DevOps
- **Timeline**: 2 years preparation + 10 months job search
- **Outcome**: $95,000 AUD DevOps Engineer at Brisbane logistics company

#### The Transformation Strategy
```markdown
## Maria's Career Pivot Approach

### Phase 1: Skill Building (18 months)
**Evening and Weekend Development**:
- Completed FreeCodeCamp full curriculum
- Built personal projects using modern tech stack
- Contributed to open source DevOps tools
- Earned multiple AWS certifications (SAA, DevOps Engineer)

**Internal Career Development**:
- Volunteered for technical projects at BPO company
- Automated manual processes using Python scripts
- Led training sessions for colleagues on technical topics
- Transitioned to technical support role within company

### Phase 2: Portfolio Development (6 months)
**Strategic Project Selection**:
- Built CI/CD pipeline for e-commerce platform with Australian payment integration
- Created infrastructure-as-code templates for AWS deployment
- Developed monitoring and alerting system for distributed applications
- Documented everything with detailed explanations and architecture decisions

**Community Involvement**:
- Joined AWS User Group Philippines and presented technical talk
- Participated in online DevOps communities
- Started technical blog documenting learning journey
- Mentored other career switchers in online forums

### Phase 3: Job Search Optimization (10 months)
**Targeted Application Strategy**:
- Researched 50 Australian companies using AWS and modern DevOps practices
- Customized applications to highlight infrastructure automation experience
- Emphasized cost optimization skills from resource-constrained BPO environment
- Positioned timezone flexibility as competitive advantage

**Interview Preparation**:
- Practiced system design interviews focusing on scalability and cost optimization
- Prepared stories about improving efficiency and reducing costs
- Studied Australian business requirements and compliance standards
- Practiced cultural adaptation for Australian communication style
```
